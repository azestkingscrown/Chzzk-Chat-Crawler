# 변경 이력 (Changelog)

이 문서는 **치지직 실시간 채팅 수집기**의 모든 주요 변경 사항을 기록합니다.

## [v1.0.0] - 2026-05-26

### ✨ 추가 (Added)
- **실시간 채팅 수집 기능 도입**: 스트리머 채널 ID를 기반으로 공식 웹소켓(WebSocket)에 연결하여 실시간으로 채팅과 후원 데이터를 수집.
- **스트리머 검색 기능**: 닉네임만 입력해도 내부적으로 네이버 치지직 검색 API를 호출하여 채널 ID를 자동으로 추출하는 로직 구현.
- **방종/뱅온 감지 기능**: 방송이 오프라인일 때 주기적(30초)으로 폴링하여 대기하다가, 방송이 켜지면 즉각적으로 웹소켓 연결을 수립하는 기능 탑재.
- **로그 저장 기능**: 콘솔 출력뿐만 아니라 `logs` 디렉토리에 텍스트 파일 형식으로 자동 영구 저장.
- **멀티 플랫폼 실행 파일 (pkg)**: Node.js가 설치되지 않은 환경(Windows, macOS, Linux)에서도 더블클릭만으로 실행되도록 `esbuild`와 `@yao-pkg/pkg`를 이용한 자동 빌드 시스템 구축.

### 🛠 수정 및 개선 (Changed & Fixed)
- **채팅 누락 버그 해결**: 치지직 웹소켓 서버의 Ping(`cmd: 0`) 신호에 주기적으로 Pong(`cmd: 10000`)으로 정확히 응답하도록 개선하여 간헐적인 강제 연결 종료 및 채팅 누락 현상 완벽 해결.
- **타임스탬프 로직 변경**: 채팅에 기록되는 시간이 현재 로컬 시간이 아닌, 해당 방송의 **실제 방송 진행 시간(경과 시간)**으로 `HH시간 MM분 SS초` 포맷으로 찍히도록 고도화.
- **이모티콘 및 특수 문자 정규화**:
  - `{:d_126:}` 같은 난해한 이모티콘 텍스트 코드를 `[이모티콘]`으로 깔끔하게 변환.
  - 텍스트 없이 단독으로 전송된 스티커(`msgTypeCode === 10` 등)가 빈 메시지로 누락되던 현상을 해결하고 정상적으로 `[이모티콘]`으로 기록되도록 처리.
- **익명 후원자 표기 변경**: 후원자 닉네임이 없을 경우 기존 '알 수 없음' 대신 사용자 친화적인 '익명의 후원자'로 표기하도록 변경.
- **실행 파일 패스워킹 최적화**: 가상 파일 시스템(`/snapshot`)에서 실행될 때 `import.meta.url` 관련 에러가 발생하는 것을 방지하기 위해 `process.cwd()` 기반의 상대 경로 로직으로 수정.
